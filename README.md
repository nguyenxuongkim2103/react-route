<!-- GETTING STARTED -->
## Getting Started

You need to install Node Package Manager first.

### Prerequisites


* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/nguyenxuongkim2103/react-route.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
